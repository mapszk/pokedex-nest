## Start

1. Clone
2. Exec `npm install`
3. Install Nest CLI
4. Hydrate DB `http://localhost:3000/api/v1/seed`
5. Start DB `docker-compose up -d`

## Environment

```
MONGODB=string
PORT=number
DEFAULT_LIMIT=number
```
